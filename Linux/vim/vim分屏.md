启动
==

        vim -On file1 file2 ... 垂直分屏

        vim -on file1 file2 ...  水平分屏

        n是数字，表示分成几个屏



关闭
==

        Ctrl + W c 关闭当前窗口

        Ctrl + W q 关闭当前窗口，如果只剩最后一个，则退出vim



分割
==

        Ctrl + W s 上下分割当前打开的文件

        ：sp filename 上下分割，并打开一个新开的文件

        Ctrl + W v 左右分割当前打开的文件

         ：vsp filename 左右分割，并打开一个新的文件



移动光标
==

        Ctrl+W l 移到右屏

        Ctrl + W h 移到左屏

        Ctrl + W k  移到上屏

        Ctrl + W j 移到下屏

        Ctrl + W w 移到下一个屏中



移动分屏
==

        Ctrl + W L 向右移动

        Ctrl + W H 向左移动

        Ctrl + W K 向上移动

         Ctrl + W J 向下移动



屏幕尺寸
==

        Ctrl + W = 让所有的屏都有一样的高度

       Ctrl + W + 增加高度

        Ctrl + W - 减少高度
